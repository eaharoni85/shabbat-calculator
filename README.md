# מחשבון שבתות - משפחת אהרוני

עמוד HTML עצמאי לחישוב סופי שבוע לפי הסכם גירושין.

נקודת הייחוס:

- 19/06/2026-20/06/2026 אצל עדי
- סוף השבוע שאחריו אצל אלירן
- לאחר מכן לסירוגין

כללי שבוע:

- יום א': עדי
- יום ב': אלירן
- יום ג': עדי
- יום ד': אלירן
- יום ה': מתחלף הפוך משישי-שבת של אותו שבוע
- יום ו' ושבת: לפי סדר השבתות המתחלף

העמוד כולל גם חריגים לחגים/תאריכים מיוחדים שנשמרים מקומית בדפדפן.

חגים אוטומטיים:

- תשפ"ו היא שנת הבסיס.
- תשפ"ז מתחלפת לפי הטבלה בהסכם.
- השנים הבאות ממשיכות לסירוגין באותו סדר.
- חנוכה נשאר לפי זמני השהות הרגילים.
- בתוך מחציות פסח וסוכות, סופי שבוע נשארים לפי סדר השבתות הרגיל.
